# Offer

大學專題

- description
  - 提供店家在平台上註冊為合作店家
  - 合作的店家會在APP上出現其廣告
  - 一般使用者可以依關鍵字或標籤搜尋到符合的店家
  - 使用者可在店家底下留言、按讚、收藏店家
  - 店家可設定簡單問答題測驗，讓使用者可以賺取折價券，可在消費時使用


- tech
  - 資料庫: firebase 
  - 前端框架 angular, ionic
